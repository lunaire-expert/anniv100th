# 100周年記念LP WordPress 導入ガイド

このディレクトリには、株式会社ルナール100周年記念LPを WordPress に導入するための完成済みパッケージが格納されています。

## 1. 配置手順

### 1-1. フォルダのアップロード
`/wp-conversion/anniv100th/` フォルダを、WordPress サーバー上の現在使用しているテーマ（または子テーマ）のディレクトリ直下にアップロードしてください。

**正しい配置パスの例:**
`.../wp-content/themes/[あなたのテーマ名]/anniv100th/`

### 1-2. テンプレートの確認
`anniv100th/` フォルダ内には以下の PHP テンプレートが含まれています。これらは WordPress の「ページテンプレート」として機能します。
- `template-v1.php` (Standard版: 光の粒子デザイン)
- `template-v2.php` (Zzz Ver: 泡のアニメーションデザイン)

## 2. WordPress 管理画面での設定

1. **固定ページの作成**: 「固定ページ」→「新規追加」を選択します。
2. **テンプレートの適用**: 画面右側の「ページ属性」セクションにある「テンプレート」ドロップダウンから以下を選択します。
   - `100th Anniv - Standard`
   - または `100th Anniv - Zzz Ver`
3. **公開**: ページを保存・公開すれば、LPが表示されます。

## 3. 注意事項

- **CSSの競合**: 本パッケージの CSS は `#anniv100th` という ID でスコープ化されており、既存テーマのスタイルと競合しにくい設計になっています。
- **functions.php**: 基本的に追記は不要ですが、特殊な環境で CSS が読み込まれない場合は、同階層の `functions-snippet.php` の内容をテーマの `functions.php` 末尾に貼り付けてください。その際、最下部の `add_action` のコメントアウトを外してください。
